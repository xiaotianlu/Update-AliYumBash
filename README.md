# Update-AliYumBash
Centos/RHEL安装阿里YUM源脚本

分图形界面版(基于Bash-Dialog),普通版,如果之前没有YUM源,使用普通版
## 安装前工作 ##
```bash
  git clone https://github.com/Mr-Linus/Update-AliYumBash  &&  cd Update-AliYumBash && chmod 755 *
```
## 图形界面安装版 ##
```bash
  sudo ./graphical-Bash.sh
```
## 普通安装版 ## 
```bash
  sudo ./common-Bash.sh
```
